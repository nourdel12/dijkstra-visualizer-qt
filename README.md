# Dijkstra Visualizer (Qt)

A C++ application built with Qt that visualizes Dijkstra’s shortest path algorithm.

## Features
- Interactive graph input
- Step-by-step visualization
- Custom priority queue implementation
- Dynamic node configuration

## Technologies
- C++
- Qt Framework
- Data Structures & Algorithms

## How to Run
1. Open `.sln` in Visual Studio
2. Build the project
3. Run using Qt environment
